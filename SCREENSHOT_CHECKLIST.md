# Screenshot Capture Checklist

Use this checklist to capture the required images for capstone submission.

## Required Screenshot Files

1. `admin_login.png` or `admin_login.jpeg`
   - Show the Django admin login page.
   - Include the URL bar with `/admin/` visible.
   - Show the `Username` and `Password` fields.

2. `admin_logout.png` or `admin_logout.jpeg`
   - Show the Django admin page after logging out.
   - Include the confirmation that the user is logged out, or the admin login page again.
   - Include the URL bar with `/admin/` visible.

3. `get_dealers.png`
   - Show the dealers on the home page of the Django application before logging in.
   - Include the browser address bar with the endpoint visible.
   - Do not show a logged-in user.

4. `get_dealers_loggedin.png`
   - Show the dealers page after logging in.
   - Include the logged-in username in the UI.
   - Show the `Review Dealer` option.
   - Include the endpoint visible in the browser address bar.

5. `dealersbystate.png` or `dealersbystate.jpeg`
   - Show dealers filtered by state on the home page.
   - Include the browser address bar with the endpoint visible.
   - Prefer filtering by `Kansas` since that is a graded requirement.

6. `dealer_id_reviews.png` or `dealer_id_reviews.jpeg`
   - Show a dealer detail page with reviews displayed.
   - Include the endpoint visible in the browser address bar.

7. `dealership_review_submission.png`
   - Show the Post Review page after entering review details, before submitting.
   - Display the form fields filled in with sample values.

8. `added_review.png`
   - Show the posted review displayed on the dealer review page.
   - Include the new review content if possible.

9. `deployed_landingpage.png`
   - Show the deployed application landing page.
   - Include the deployment URL in the browser address bar.

10. `deployed_loggedin.png`
    - Show the deployed application after login.
    - Include the username in the UI.
    - Include the deployment URL in the browser address bar.

11. `deployed_dealer_detail.png`
    - Show the deployed dealer details page.
    - Include the endpoint or URL bar.

12. `deployed_add_review.png`
    - Show the deployed review displayed in the deployed application.
    - Include the deployment URL or endpoint visible.

## Tips for capture

- Use a browser screenshot tool or capture key portions of the screen.
- Make sure the URL bar is visible in each screenshot where required.
- Use a desktop browser rather than a mobile layout.
- Keep text legible and avoid cropping out form fields or buttons.
- Save images with the exact required file names.
